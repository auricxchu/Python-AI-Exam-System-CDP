# 验证清单

## 环境准备

- Win11 小屏幕（1366x768），建议外接显示器拔掉只用笔记本屏幕
- 清除 `localStorage` 中 `app_opening_seen_v2` 键（模拟首次打开）
- 确保网络正常，Supabase 后端可用

---

## 1. 开机动画

- [ ] **首次打开**：播放完整 full 版动画（代码逐字输入效果），无闪烁、无代码跳动
- [ ] **关闭重开**：第二次打开播放 lite 版（只显示 logo + 渐变消失），不重复 full 版
- [ ] **动画异常恢复**：如果动画中途出错，下次打开仍走 lite 版（不会卡在 full 版循环）

## 2. 小屏幕自适应（1366x768）

- [ ] **主页**：所有元素（标题、图标、卡片、模型选择器、退出按钮、版权文字）完整可见，不需滚动
- [ ] **成绩单**：页面可滚动到底部，分数和标题字号适当缩小，不会挤满屏幕
- [ ] **教师后台**："云端题库管理"标题字号适当

## 3. 控制台多行 print

- [ ] 进入考试，在编辑器中写：
  ```python
  print("第一行")
  print("第二行")
  print("第三行")
  ```
- [ ] 点击运行，控制台正确显示全部三行，不丢失、不重复

## 4. IME Pill（输入法状态指示器）

- [ ] **中文输入法**切换为中文模式时，pill 显示"中"
- [ ] **切换为英文模式**时，pill 显示"英"
- [ ] **CapsLock** 开关时，pill 上 CAPS 图标正确切换（锁/解锁）
- [ ] **切换输入法类型**（微软拼音 → 搜狗/百度等）时 pill 不闪、不卡死
- [ ] **IME helper 未启动**时 pill 不显示（不出现空白或乱码）

## 5. 离线考试安全

- [ ] **正常网络**：学生登录时通过网络检测，正常进入考试
- [ ] **Supabase 不可用**：学生登录时被拦截，提示网络检测失败
- [ ] **考试中断网**：
  - [ ] 断网后顶栏显示断网图标（WifiOff）
  - [ ] 60 秒后弹出全屏遮罩"网络连接已断开，考试已暂停"
  - [ ] 锁定期间无法编辑代码、无法运行
- [ ] **恢复网络**：重新联网后遮罩自动消失，可继续答题
- [ ] **交卷后断网**：已完成考试后断网不会触发锁定

## 6. 回归验证（确保 Windows 特有功能未受影响）

- [ ] **反馈功能**：成绩单页面可提交反馈
- [ ] **Kiosk 安全模式**：考试期间 Alt+Tab/ Win 键被拦截（如有实现）
- [ ] **本地资源加载**：离线资源（本地 Pyodide/Monaco）优先加载正常
- [ ] **教师后台云端同步**：保存/加载题库正常
